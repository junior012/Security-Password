Quais as características de uma senha forte?

Longa

Quanto mais longa a senha, mais segura ela é. Senhas devem ter pelo menos 10 caracteres.

Complexa

Uma senha forte é composta por uma combinação de letras maiúsculas, minúsculas, números e símbolos.

Única

Utilize senhas diferentes para cada conta para minimizar riscos em caso de vazamento de dados.

Lembre-se de usar um gerenciador de senhas para armazenar todas as suas senhas de forma segura.
